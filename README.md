# مشروع Blender: كرة تدمر مربع

هذا مشروع صغير باستخدام Blender يوضح حركة كرة تدمر مربعًا بشكل واقعي باستخدام المحاكاة الفيزيائية.

## وصف المشروع
- تم إنشاء مشهد يحتوي على كرة ومربع.
- عند تشغيل المحاكاة، تتحرك الكرة وتصطدم بالمربع مما يؤدي إلى تكسيره.
- المشروع يستخدم أدوات الفيزياء في Blender مثل **Rigid Body** و**Collision** لتجربة حركة طبيعية.

## محتويات المشروع
- ملف Blender (.blend) للمشروع
- ملفات إضافية إذا وجدت (صور، نصوص، مواد، الخ)

## كيفية التشغيل
1. افتح الملف في Blender.
2. انتقل إلى **Timeline** واضغط على **Play** لمشاهدة الحركة.
3. يمكن تعديل خصائص الكرة أو المربع للحصول على تأثيرات مختلفة.

## ملاحظات
- المشروع تعليمي ويهدف لفهم أساسيات المحاكاة الفيزيائية في Blender.
- يمكن تطويره لاحقًا بإضافة المزيد من الأجسام أو المؤثرات البصرية.
